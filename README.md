# Autonomous Trading Agent Lab

This repository contains research notes, design documents, and CLI-first prototypes for studying autonomous trading agents.

The main goal is to build strong domain understanding and a cost-aware architecture before implementing complex systems.

## Project Focus

- Autonomous trading agents
- Financial market simulation
- Time-series and tabular data infrastructure
- Agent architecture for trading workflows
- Feature engineering, labeling, and validation
- Cost-vs-efficiency tradeoffs when using LLMs
- CLI-first research and prototyping

## Current Stage

This repository is currently in the research and architecture design phase.

The priority is not to build a full backend or frontend immediately. Instead, the project focuses on:

1. Building domain knowledge
2. Reviewing relevant papers and frameworks
3. Designing a practical agent pipeline
4. Preparing small scripts for reproducible experiments
5. Evaluating when LLMs are useful and when simpler methods are better

## Design Principles

- CLI-first before UI
- Docs-first before implementation
- Avoid unnecessary LLM usage
- No LLM fine-tuning in the early stage
- Prefer useful, measurable, and cost-aware components
- Keep public repository free of private data, raw papers, credentials, and company-sensitive materials
