# Project Docs Overview

This section stores documentation about the actual project.

## What belongs here?

- Project goals
- Architecture
- Roadmap
- CLI design
- Experiment design
- Task logs
- Technical decisions

## Current Direction

The project focuses on autonomous trading agents / financial AI agents.

The first stage prioritizes:

- Domain understanding
- Architecture design
- CLI-first prototype
- Cost-vs-efficiency analysis
- Evaluation methodology

Backend, frontend, and deployment are not the main focus in the early stage.
