# Roadmap

## Phase 1: Domain Knowledge

- Review papers and reports
- Understand autonomous trading agents
- Study financial machine learning concepts
- Study labeling, feature engineering, validation, and backtesting

## Phase 2: Architecture Design

- Define the trading agent pipeline
- Design data flow
- Define agent roles
- Define evaluation methodology

## Phase 3: CLI Prototype

- Build small reproducible scripts
- Run simple data, feature, and backtest experiments
- Log outputs clearly

## Phase 4: Demo Packaging

- Add minimal UI/backend only if needed
- Keep the project reviewable and easy to run
